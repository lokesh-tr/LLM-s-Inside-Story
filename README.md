# LLM's Inside Story
Swift Student Challenge 2024 – Winner

This repository contains the exact code submitted for Apple Swift Student Challenge 2024, and is archived.

Developed **LLM’s Inside Story**, an interactive SwiftUI educational app that teaches how Large Language Models work through child-learning analogies and gamified experiences.
- Built complete MVVM architecture using SwiftUI, Combine for reactive state management, CoreGraphics for token visualizations, and spring animations
- Implemented 4-step progressive tutorial, draggable TokenBlock components, 3-question merge-rule quiz with real-time scoring, with accessibility.
- Delivered polished, delightful iOS app that achieved global Apple Winner recognition

**Skills:** Swift, SwiftUI, Combine, CoreGraphics, Xcode, Figma, iOS Development
